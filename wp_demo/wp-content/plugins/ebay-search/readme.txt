
Installation 

1) extract zip to wp-content\plugins

2) activate plugin from admin > plugins

3) goto admin > settings > ebay API 

4) set values for appID, devID, certID,...

4) add shorttag [ebay] to page in which you want to display ebay products 

If you have not your own devID, certId, appID,.. etc then you can use following values
